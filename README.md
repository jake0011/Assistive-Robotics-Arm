# Assistive-Robotics-Arm
This project involves designing and building a table-top assistive robotic arm to help individuals with limited mobility pick and place small household objects. The arm will be controlled through voice commands or a simple joystick, using Arduino for motor control and a Raspberry Pi for speech recognition and processing.

## Tech Stack

- **Arduino Uno R3** — Servo motor control via PWM
- **Raspberry Pi** — Offline speech recognition (Vosk)
- **Python 3.7+** — Voice processing and serial communication
- **pySerial** — Raspberry Pi to Arduino communication
- **Vosk** — Offline speech recognition engine
